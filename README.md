# タガログ語 2ヶ月チャレンジ

Riri先生のタガログ語テキスト(全31課)をもとにした、2ヶ月間のタガログ語学習アプリです。

## 内容

- **単語フラッシュカード**: SM-2式の間隔反復で効率よく暗記
- **文法クイズ**: 解禁済みレッスンの単語・例文から自動生成される4択問題
- **会話練習**: 日本語 → タガログ語の瞬発力トレーニング
- **レッスン一覧**: 各課の文法ノート・単語・例文をいつでも参照可能

全32レッスン(単語533語・例文303文)を、1〜47日目にかけて約1.5日に1課ずつ解禁し、残りの期間は総復習にあてる設計になっています。

## 使い方

`index.html` をブラウザで開くだけで動作します(サーバー不要、外部通信なし)。進捗はブラウザのlocalStorageに保存されます。

GitHub Pagesで公開している場合は、そのURLをスマホのホーム画面に追加して毎日開く使い方がおすすめです。
